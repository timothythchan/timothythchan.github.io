A well-tuned development environment can dramatically improve your productivity and enjoyment.
Here's a look at the setup I use every day.

## Editor

**VS Code** is my primary editor. Key extensions I rely on:
- **ESLint** + **Prettier** — consistent code style
- **GitLens** — supercharged Git integration
- **Tailwind CSS IntelliSense** — autocomplete for utility classes
- **Error Lens** — inline error display

My theme: **One Dark Pro** with **JetBrains Mono** font at 14px.

## Terminal

**iTerm2** (macOS) with **Zsh** + **Oh My Zsh**. Useful plugins:
- `git` — Git aliases
- `zsh-autosuggestions` — fish-like suggestions
- `zsh-syntax-highlighting` — color-coded commands

## CLI Tools

```bash
# Navigation
z / zoxide    # Smart directory jumping
fzf           # Fuzzy file/history finder
bat           # Better cat with syntax highlighting
eza           # Better ls with icons

# Development
gh            # GitHub CLI
httpie        # Friendly HTTP client
jq            # JSON processor
```

## Version Management

- **nvm** for Node.js
- **pyenv** for Python
- **Docker** for everything else

- **MacBook Pro** (Apple Silicon) — my primary machine
- **External Monitor** — more screen real estate matters
- **Mechanical Keyboard** — [Your keyboard here]

- **Raycast** — launcher + snippets + clipboard history
- **Obsidian** — personal notes and knowledge base
- **Linear** — project tracking
